File Organization Principles

  • Components: One component per file, named after the component

  • Utilities: Create /src/utils/ when needed for helper functions

Component Structure

Standard Component Template 
import React, { useState, useEffect, useRef } from "react";
import { Link } from "react-router-dom";

export default function ComponentName({ theme, data }) {
    // 1. State declarations
    const [isOpen, setIsOpen] = useState(false);
    
    // 2. Refs
    const dropdownRef = useRef(null);
    
    // 3. Effects
    useEffect(() => {
        // Effect logic
        return () => {
            // Cleanup
        };
    }, [dependencies]);
    
    // 4. Event handlers
    const handleClick = () => {
        setIsOpen(!isOpen);
    };
    
    // 5. Render
    return (
        <div className="...">
            {/* JSX */}
        </div>
    );
}

Naming Conventions

Good PascalCase

ComponentName.jsx
export default function ComponentName() { ... }

Good camelCase

const userName = "Lucas";
const handleSubmit = () => { ... };

Good SCREAMING_SNAKE_CASE (when truly constant)

const MAX_RETRIES = 3;
const API_BASE_URL = "https://api.example.com";

Props Patterns

Good Destructure props in function signature:

export default function Card({ title, description, theme }) {
    return <div>...</div>;
}

Avoid Using props object:

export default function Card(props) {
    return <div>{props.title}</div>;
}

üìú JavaScript/JSX Best Practices

Modern JavaScript

const/let, Arrow Functions, Destructuring

    console.log("clicked");

JSX Guidelines

Good Always use keys in lists:

{items.map(item => (
    <div key={item.id}>{item.name}</div>
))}

Avoid Using index as key (unless order never changes):

{items.map((item, index) => (
    <div key={index}>{item.name}</div>
))}

‚ö° Performance Patterns

Avoid Unnecessary Re-renders

useMemo & useCallback 
import { useMemo, useCallback } from "react";

// Memoize expensive computations
const sortedItems = useMemo(() => {
    return items.sort((a, b) => a.name.localeCompare(b.name));
}, [items]);

// Memoize callbacks passed to children
const handleClick = useCallback(() => {
    doSomething(value);
}, [value]);

Code Splitting

Lazy Load Routes 
import { lazy, Suspense } from "react";

const Home = lazy(() => import("./pages/Home"));
const About = lazy(() => import("./pages/About"));

<Suspense fallback={<Loading />}>
    <Routes>
        <Route path="/" element={<Home />} />
        <Route path="/about" element={<About />} />
    </Routes>
</Suspense>

Common React Patterns

Click Outside Detection 
useEffect(() => {
    function handleClickOutside(event) {
        if (ref.current && !ref.current.contains(event.target)) {
            setIsOpen(false);
        }
    }
    document.addEventListener("mousedown", handleClickOutside);
    return () => document.removeEventListener("mousedown", handleClickOutside);
}, []);
Local Storage Persistence 
const [value, setValue] = useState(() => {
    const stored = localStorage.getItem('key');
    return stored ? JSON.parse(stored) : defaultValue;
});

useEffect(() => {
    localStorage.setItem('key', JSON.stringify(value));
}, [value]);
